# 🧠 Quiz App

A simple and interactive **Quiz Application** built using **HTML, CSS, and JavaScript**.  
This app allows users to answer multiple-choice questions and displays their score at the end.

---

## 🚀 Features
- ❓ Multiple-choice questions  
- 🔘 Select one answer per question  
- ⏭️ Next question navigation  
- 📊 Score calculation  
- ⚡ Instant result display  
- 🎨 Clean and simple UI  
- 📱 Responsive layout  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## ▶️ How to Run
1. Download or clone this repository  
2. Open **quizz.html** in your browser  

---

## 📸 Overview
This app allows you to:
- Answer quiz questions one by one  
- Select options using radio buttons  
- Navigate through questions  
- View your final score at the end
